# 🍌 Minh Hub - Blox Fruits Script

Minh Hub is a custom script for Blox Fruits that provides a clean and user-friendly UI with useful automation features.

## 🚀 Features
- Auto Farm
- Auto Skills (Z, X, C, V)
- Auto Click
- Simple and clean Banana-style UI
